# Experiment No: 1e – SEB-Simple Interest Calculation in Python

## AIM  
To develop a Python program that accepts principal amount, rate of interest, and time, and calculates the simple interest using the standard mathematical formula.

## ALGORITHM  
1.Start the program.
2.Read three float inputs: a (principal), b (rate), and c (time).
3.Compute simple interest using the formula: s = (a * b * c) / 100.
4.Print the result formatted to two decimal places.
5.End the program.

## PROGRAM
```python
a=float(input())
b=float(input())
c=float(input())
s=(a*b*c)/100
print("Simple interest : %0.2f"%s)
```
